# 人工智能信息雷达分析助手

> 帮助学生与 AI 从业者从海量信息中高效学习人工智能前沿动态，支持 AI 周期简报与深度话题调研两种模式，自动追踪黑客松活动并解释 AI 新名词的 Skill。

![License](https://img.shields.io/badge/license-自定义版权-red) ![Platform](https://img.shields.io/badge/platform-Claude%20Code-blue) ![Skill](https://img.shields.io/badge/type-Skill-purple)

## 简介

这个技能帮助学生和 AI 从业者从海量信息中**高效学习**人工智能领域的前沿动态，支持两种工作模式：

- **模式一：AI 周期简报** - 自动生成日报/周报，覆盖热点新闻、技术突破、新概念解释、论文速览、黑客松与活动日历
- **模式二：深度话题调研** - 针对指定 AI 话题，多源搜索并输出结构化综述报告（附概念解释）

**特别设计**：每期简报都会标注本周出现的 AI 新名词并附通俗解释，同时追踪全球黑客松活动，方便找到动手实践的机会。

## 功能特性

- 📰 **五类信息多源搜索**：新闻热点、技术突破、新概念速递、论文速览、黑客松与活动
- 🧠 **AI 新名词通俗解释**：每个新术语附「全称 + 通俗解释 + 为什么火 + 延伸阅读」
- 🏆 **黑客松活动追踪**：日期 + 地点 + 奖金 + 适合人群 + 报名链接
- 🔬 **深度话题调研**：技术/学术/产业/政策/竞品五维度综述
- ✅ **交叉验证**：多源印证、信源可信度标注、待验证信息明确标识

## 适用场景

- 学生持续学习 AI 前沿知识、理解新概念、寻找比赛机会
- AI 开发者寻找黑客松和开源项目锻炼技能
- AI 行业从业者持续跟踪前沿动态
- 研究者快速了解某子领域最新进展

## 快速开始

### 1. 克隆仓库到 Claude Code 的 skills 目录

```bash
git clone https://github.com/landawang-star/ai-xinxileida-fenxi.git ~/.claude/skills/人工智能信息雷达分析助手
```

### 2. 部署到你的 Agent

详见 [部署指南.md](./部署指南.md)，支持 Claude Code、Codex、Cursor、Windsurf、Cline、Gemini CLI 等。

### 3. 开始使用

在 Agent 对话中使用以下触发词，或直接描述需求：

- AI行业周报
- 人工智能最新动态
- 生成AI简报
- AI前沿调研
- AI新名词解释
- 人工智能一周热点
- 帮我找AI最新论文
- 有什么AI比赛
- 黑客松在哪里举办
- 近期AI活动有哪些
