# AI 信息源速查手册

## 搜索策略

### 通用搜索技巧
- 中文搜索：优先使用百度、知乎、微信公众号、机器之心、量子位
- 英文搜索：优先使用 Google、Twitter/X、Reddit、Hacker News
- 论文搜索：arXiv、Papers With Code、Semantic Scholar、Google Scholar
- 时效性要求高的：限定近 7 天/近 1 个月
- 深度调研：扩大至近 3-6 个月

### 关键词组合策略
不同 AI 子领域的搜索关键词建议：

| 领域 | 中文关键词 | 英文关键词 |
|------|-----------|-----------|
| 大语言模型 | LLM、大模型、GPT、Claude、Gemma、开源大模型 | LLM, GPT, Claude, open-source LLM |
| 多模态 | 多模态、视觉语言、文生图、文生视频 | multimodal, vision-language, text-to-image |
| AI Agent | 智能体、AI Agent、自主Agent、工具调用 | AI agent, autonomous agent, tool use |
| 具身智能 | 机器人、具身智能、人形机器人 | embodied AI, robotics, humanoid |
| AI 安全 | AI安全、对齐、幻觉、越狱 | AI safety, alignment, hallucination |
| AI 应用 | AI应用、AI编程、AI设计、AI教育 | AI application, copilot, AI tools |

## 权威信息源分级

### S 级（官方一手信息，最可信）
- OpenAI Blog / Anthropic Blog / Google DeepMind Blog / Meta AI Blog
- arXiv 论文原文
- 官方 GitHub 仓库 release notes
- 政府部门 AI 政策文件

### A 级（专业媒体报道，较可信）
- TechCrunch / The Verge / Wired
- 机器之心 / 量子位 / 极客公园
- Nature / Science 科技新闻版块

### B 级（社区讨论，需交叉验证）
- Hacker News / Reddit (r/MachineLearning)
- Twitter/X 上知名 AI 研究者账号
- 知乎 AI 话题精华
- 微信公众号 AI 相关账号

### C 级（传闻/推测，必须标注"待验证"）
- 匿名爆料
- 未证实的产品发布传闻
- 社交媒体上非权威账号推测

## 信息来源核实方法

1. **论文**：必须有 arXiv ID 或 DOI，可追溯到原文
2. **产品发布**：必须有官方博客或发布会链接
3. **投融资**：必须有工商信息或可信科技媒体报道
4. **基准测试**：必须有排行榜链接（如 Chatbot Arena、Open LLM Leaderboard）
5. **活动信息**：必须有官网或官方社交媒体公告

---

## 时效性判断

- 标注每条信息的发布日期
- 对超过 1 个月的信息：标注"已发布 X 天，可能已有更新进展"
- 周报中优先使用近 7 天信息
- 深度调研可包含更早的里程碑事件作为背景
